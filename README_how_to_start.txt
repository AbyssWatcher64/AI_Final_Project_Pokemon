=================
Steps for testing
=================
1. Download mGBA: https://mgba.io/downloads.html
2. Download and install python: https://www.python.org/downloads/ 
3. Load ROM to mGBA:  Rom_and_Saves/Pokemon - Emerald Version (U).gba
4. Load the state file: Rom_and_Saves/Pokemon - Emerald Version (U).ss1
5. Tools -> Scripting -> File -> Load Script... -> control_server.lua
6. Open the file mgba_controller.py with Python 3.13 or greater (lower versions might work too).
7. Select mode 2. Start inputting the keys as stated by the python script.
