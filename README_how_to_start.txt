=================
Steps for testing
=================
1. Download mGBA Development version: https://mgba.io/downloads.html#development-downloads
2. Download and install python: https://www.python.org/downloads/ 
3. Load ROM to mGBA:  Rom_and_Saves/pokemon_emerald.gba
4. Load the state file: Rom_and_Saves/pokemon_emerald.ss1
5. Tools -> Scripting -> File -> Load Script... -> mgba_environment_server.lua
6. Open the file mgba_environment_client.py with Python 3.13 or greater (lower versions might work too).
7. Select mode 2. Start inputting the keys as stated by the python script.
